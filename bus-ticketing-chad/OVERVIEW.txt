🚀 QUICK START (5 MINUTES)

1. Install dependencies
   $ cd backend && npm install
   $ cd frontend && npm install

2. Setup database
   $ createdb bus_ticketing_chad
   $ cd backend && npx prisma migrate dev

3. Seed data
   $ npm run seed

4. Start servers
   $ npm run dev (from root)

5. Access
   → Frontend: http://localhost:3000
   → Backend: http://localhost:5000
   → Admin: +23566778899 (OTP in console)

💡 KEY DESIGN DECISIONS

❓ Why OTP instead of passwords?
   → Phone numbers universal in Chad
   → WhatsApp/SMS more accessible
   → Simpler for users

❓ Why manual payment confirmation?
   → Mobile Money APIs vary
   → Network issues common
   → Gives admins control

❓ Why WhatsApp priority?
   → Most used app in Chad
   → More reliable than SMS
   → Can send rich media

❓ Why guest bookings?
   → Lower barrier to entry
   → Phone number sufficient
   → Can convert later
